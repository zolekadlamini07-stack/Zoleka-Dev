# Zoleka-Dev

A living showcase of concept understanding through practical examples.

This repository is organised by topic area. Each folder contains documented examples with code in C# and TypeScript where applicable. This is a growing repository — the changelog tracks what has been added and when.

## Structure

| Folder | Contents |
|---|---|
| oop/ | Object-Oriented Programming concepts using a finance domain |
| clean-architecture/ | Architecture principles and patterns |
| cloud-native/ | Azure services and cloud-native patterns |
| tdd-and-katas/ | Test-driven development and coding katas |
| react-typescript/ | React and TypeScript fundamentals |
| project-concepts/ | Larger architectural concepts demonstrated through working mini-projects |

## Stack

- Backend: C# / .NET, Clean Architecture, Wolverine, Entity Framework
- Frontend: React, TypeScript, Vite
- Cloud: Azure App Configuration, Service Bus, Azure Functions, Aspire
- Testing: xUnit, Vitest, React Testing Library
