# Object-Oriented Programming

Core OOP concepts demonstrated in C# and TypeScript using a finance domain. Finance was chosen because it maps naturally to every OOP principle and mirrors real-world domain logic.

Each concept folder contains a README explaining the concept, and code examples in both languages.
